What is Dwaarai?

Dwaarai is a cloud-managed, edge-powered vehicle access control platform built for gated residential communities. It replaces paper logbooks, manual boom barrier operation, and unreliable intercom systems with fully automated gate entry.

A registered vehicle with a FASTag drives up → the gate opens automatically in under 500 milliseconds — no stop, no wait, no guard interaction.

Core Capabilities

🚗

FASTag Auto-Entry

UHF RFID reads the vehicle's existing FASTag at 2-5 metre range. Known vehicles get instant gate-open (<500ms). New FASTags are auto-paired on first drive-through.

📷

ANPR Camera

Automatic Number Plate Recognition captures every vehicle's plate for verification, auto-pairing, and a complete visual audit trail with timestamped snapshots.

👤

Visitor Management

Residents create time-bound visitor passes with OTP codes. Share via WhatsApp. Guards verify at gate. Residents get push notifications to approve or deny.

🔒

Offline Resilient

Edge nodes maintain a local whitelist. Known vehicles enter even without internet. Events queue locally and sync to cloud when connectivity restores.

How It Works

Resident Vehicle Entry (Automatic)

1

Vehicle Approaches Gate

UHF RFID reader detects the vehicle's FASTag at 2–5m range. No stopping required.

2

Instant Verification

Edge controller checks the FASTag against the local whitelist. Match found → gate opens in under 500ms.

3

ANPR Captures Plate

Camera simultaneously captures the licence plate for audit records and a visual snapshot.

4

Resident Notified

Push notification sent to the resident's phone: "Your KA05MF1234 entered Main Gate at 8:42 AM".

First-Time FASTag Pairing (Zero Setup)

Residents don't need to "register" their FASTag. They simply register their vehicle plate. On the first drive-through, the system auto-detects and links the FASTag — all future entries are instant.

1

Unknown FASTag Detected

UHF reader picks up a new FASTag not yet in the system.

2

ANPR Matches Registered Plate

Camera reads the plate and matches it to a registered vehicle in the community.

3

Auto-Pair & Open

FASTag is permanently linked to the vehicle. Gate opens. Resident receives a confirmation: "FASTag linked — auto-entry active!"

Visitor Entry

Resident Creates Pass

  • Opens Dwaarai app → Visitors tab
  • Enters visitor name, validity, max visits
  • System generates a 6-digit OTP
  • Shares OTP via WhatsApp / SMS

Visitor Arrives at Gate

  • Provides OTP to guard
  • Guard verifies on tablet → gate opens
  • Or: Guard taps "Notify Resident"
  • Resident gets push notification to approve / deny

Cloud Platform

Hosted on AWS. Manages all communities, vehicles, visitors, and gate commands centrally. Provides REST API, WebSocket real-time updates, MQTT messaging to edge nodes, and push notification delivery. PostgreSQL database with monthly-partitioned audit logs.

Node.js PostgreSQL Redis MQTT AWS
⚙️

Edge Controller

Runs on a mini-PC at each gate. Interfaces with the ZKTeco C3-100 access controller, UHF RFID reader, and ANPR camera. Makes local access decisions using a cached whitelist. Queues events when offline and syncs to cloud on reconnect.

Python SQLite MQTT Offline-First
